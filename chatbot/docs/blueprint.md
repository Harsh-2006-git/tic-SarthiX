# **App Name**: RoamAI

## Core Features:

- AI Travel Agent: Uses the Gemini 1.5 Flash model to interview the user about their travel preferences (Destination, Duration, Budget, and Style) using a .prompt file and then generates a structured JSON Itinerary using Firebase Genkit.
- Itinerary Persistence: Saves the generated trip itinerary to Firestore, associated with the user authenticated via Firebase Auth.
- Chat Interface: A chat UI built with React and Tailwind CSS on the left side of the screen for interacting with the AI travel agent, using the httpsCallable function to call the Genkit flow.
- Visual Itinerary Display: Displays the structured JSON Itinerary generated by the AI on the right side of the screen, with a 'Loading' state indicated by a progress bar or skeleton cards during the AI's generation process. Uses lucide-react icons.
- Attraction Search Tool (Mock): A mock tool searchNearbyAttractions is provided for the AI tool to gather data for the itinerary, and improve the itinerary generation with more grounded tool based context. The LLM uses this tool when the tool is helpful.

## Style Guidelines:

- Primary color: Soft teal (#64CCC9) evokes tranquility and discovery.
- Background color: Light grayish-teal (#E5F4F4) provides a calm, unobtrusive backdrop.
- Accent color: Muted lime green (#A1E44A) is used sparingly to highlight interactive elements.
- Headline font: 'Poppins', a geometric sans-serif for a contemporary and fashionable feel; Body font: 'PT Sans', a humanist sans-serif, providing a balance of modernity and readability.
- Utilize consistent and clear icons from lucide-react to represent different aspects of the itinerary.
- Employ a split-screen layout with the chat UI on the left and the visual itinerary on the right to maintain a balanced user experience.
- Use subtle animations and transitions to enhance user engagement and provide feedback during loading states.
